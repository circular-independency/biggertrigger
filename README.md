# TriggerRoyale

A multiplayer game similar to laser tag, but you play it with your phone.

Players join a lobby and use their phone's camera to spot and "tag" each other. No special equipment needed, just your phone.

## What's in this repo

- `flutter_app/` — the mobile app (Flutter) with the menu, lobby and game UI.
- `TriggerRoyale/` — the Android vision module that handles camera detection.
- `socket_server/` — the Python server that connects players together.
- `3D_gun/` — optional 3D-printable phone mount that looks like a gun.
